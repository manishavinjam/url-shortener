\# 🔗 URL Shortener with Analytics



A full-stack URL Shortener web application built with Java Spring Boot that allows users to shorten long URLs and track click analytics in real time.



\## 🚀 Live Features



\- ✅ Shorten any long URL instantly

\- ✅ Redirect to original URL via short link

\- ✅ Track click count per URL

\- ✅ Real-time analytics dashboard

\- ✅ Clean and responsive UI



\## 🛠️ Tech Stack



| Layer     | Technology            |

|-----------|-----------------------|

| Backend   | Java Spring Boot      |

| Database  | H2 In-Memory Database |

| ORM       | Spring Data JPA       |

| Frontend  | HTML, CSS, JavaScript |

| Build Tool| Maven                 |

\## ⚙️ How to Run Locally



1\. Clone the repository

```bash

git clone https://github.com/manishavinjam/url-shortener.git

```



2\. Open in Eclipse IDE



3\. Run as Spring Boot App



4\. Open browser and go to:  http://localhost:8080/index.html



\## 📊 API Endpoints



| Method | Endpoint       | Description                  |

|--------|----------------|------------------------------|

| POST   | /shorten       | Shorten a URL                |

| GET    | /r/{shortCode} | Redirect to original URL     |

| GET    | /analytics     | Get all URLs with click data |
