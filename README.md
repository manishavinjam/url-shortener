# 🔗 URL Shortener with Analytics

A full-stack web app to shorten URLs and track click analytics in real time.

🌐 **Live Demo:** [Click Here](https://url-shortener-production-7575.up.railway.app/index.html)

---

## ✨ Features
- Shorten any long URL instantly
- One-click redirect via short link
- Real-time click count tracking
- Analytics dashboard with all URLs

---

## 🛠️ Tech Stack
- **Backend:** Java Spring Boot
- **Database:** H2 In-Memory
- **Frontend:** HTML, CSS, JavaScript
- **Deployed on:** Railway

---

## 🚀 Run Locally

git clone https://github.com/manishavinjam/url-shortener.git
cd url-shortener
mvnw.cmd spring-boot:run

Open → http://localhost:8080/index.html
